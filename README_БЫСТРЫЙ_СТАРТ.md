# Быстрый старт: работа без бота

## Запуск за 3 шага

### Шаг 1. Установите зависимости (если еще не установлены)
```bash
cd /Users/kelemetovmuhamed/Documents/vpnwebsite
npm install
```

### Шаг 2. Запустите сайт
```bash
npm run dev
```

### Шаг 3. Откройте в браузере
Откройте: **http://localhost:5173**

Нажмите кнопку **"Продолжить с Telegram"** — авторизация произойдет автоматически с моковыми данными.
