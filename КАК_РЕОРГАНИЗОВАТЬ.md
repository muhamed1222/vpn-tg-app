# Инструкция по реорганизации проекта

## Задача
Переместить старый проект в папку `old` и сделать `outlivion-miniapp` главным проектом.

## Шаги выполнения

### 1. Откройте терминал и перейдите в папку проекта:
```bash
cd /Users/kelemetovmuhamed/Documents/vpnwebsite
```

### 2. Создайте папку `old`:
```bash
mkdir -p old
```

### 3. Переместите все старые файлы и папки в `old`:
```bash
# Переместить файлы
mv App.tsx constants.tsx eslint.config.js favicon.svg index.css index.html index.tsx metadata.json playwright.config.ts prettier.config.cjs tailwind.config.cjs tsconfig.json types.ts vite.config.ts old/

# Переместить папки
mv components context dist e2e fonts hooks pages public services test test-results utils vpnwebsite old/

# Переместить документацию и скрипты
mv *.md *.sh *.txt *.cjs old/ 2>/dev/null

# Переместить package.json
mv package.json package-lock.json old/
```

### 4. Скопируйте содержимое `outlivion-miniapp` в корень:
```bash
cp -r outlivion-miniapp/* .
cp outlivion-miniapp/.eslintrc* . 2>/dev/null
```

### 5. Удалите папку `outlivion-miniapp`:
```bash
rm -rf outlivion-miniapp
```

### 6. Удалите временные скрипты:
```bash
rm -f reorganize.sh reorganize_final.py REORGANIZE_INSTRUCTIONS.md DO_REORGANIZE.sh FINAL_REORGANIZE.sh КАК_РЕОРГАНИЗОВАТЬ.md
```

## Проверка результата

После выполнения команд проверьте:

```bash
# Должен быть новый package.json с "outlivion-miniapp"
cat package.json | grep "outlivion-miniapp"

# Должна быть папка app
ls -d app

# Должна быть папка old со старыми файлами
ls old/
```

## Итоговая структура должна быть:

```
vpnwebsite/
├── app/                    # ✅ Новый проект (Next.js)
├── components/             # ✅ Новый проект
├── lib/                    # ✅ Новый проект
├── store/                  # ✅ Новый проект
├── package.json            # ✅ Новый проект (outlivion-miniapp)
├── next.config.ts          # ✅ Новый проект
└── old/                    # ✅ Старый проект
    ├── App.tsx
    ├── components/
    ├── pages/
    ├── package.json
    └── ...
```
