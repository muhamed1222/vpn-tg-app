# Краткая настройка бота @Test_outlivion_bot

## Данные бота

- **Username:** `@Test_outlivion_bot`
- **Токен:** `ваш_токен_бота`

## Быстрый старт

### Шаг 1. Сайт (.env.local)
```bash
echo "VITE_API_URL=http://localhost:3000" > .env.local
```

### Шаг 2. Бот (.env)
```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
TELEGRAM_USE_POLLING=1
PORT=3000
ALLOWED_ORIGINS=http://localhost:5173,http://localhost:3000,https://web.telegram.org
```

### Шаг 3. Запуск
```bash
# Терминал 1
cd /path/to/vpn_bot && npm run dev

# Терминал 2
cd /Users/kelemetovmuhamed/Documents/vpnwebsite && npm run dev
```

### Шаг 4. Настройка WebApp
1. @BotFather → `/mybots` → `@Test_outlivion_bot`
2. `Bot Settings` → `Menu Button` → Web App
3. URL: `https://ваш-туннель.ngrok.io` (для локальной разработки)

### Шаг 5. Тестирование
1. Откройте https://t.me/Test_outlivion_bot
2. `/start`
3. Нажмите на кнопку WebApp
4. Нажмите "Продолжить с Telegram"
